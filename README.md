<h1 align="center"> IDEF diagrams </h1>
<p align="center"><b>Практические и лабораторные по IDEF diagrams </b></p>
<ul>
<li>Методологии семейства ICAM для решения задач моделирования сложных систем, позволяют отображать и анализировать модели деятельности широкого спектра сложных систем в различных разрезах. При этом широта и глубина обследования процессов в системе определяется самим разработчиком, что позволяет не перегружать создаваемую модель излишними данными</li>
<p align="center"><b>Инструкция по открытию IDEF diagrams</b></p>
<ul>
  <li> Выбрать ветку </li>
  <li> Выбрать диаграмму </li>
  <li> Нажимаем кнопку "Raw" </li>
  <li> Нажимаем на правую кнопку мыши </li>
  <li> В открывшемся окошке нажимаем "Сохранить как" </li>
  <li> Скачать диаграмму на устройство </li>
  <li> Открыть сайт diagrams.net </li>
  <li> Нажать на кнопку "Файл" </li>
  <li> Нажать на кнопку "Открыть из" </li>
  <li> В появившемся списке нажимаем кнопку "Это устройство..." </li>
  <li> Выбираем файл с диаграммой </li>
